To run the code, just compile with:

    swipl -o solveKojun -c solveKojun.pl

Execute with:

    ./solveKojun.exe - windows
    ./solveKojun - linux

From the terminal, after execution:

    ?- solve("input.txt").  

The input files and refering puzzles are:

    - 8.txt  = https://www.janko.at/Raetsel/Kojun/009.a.htm

    - 10.txt = https://www.janko.at/Raetsel/Kojun/010.a.htm
